# KTL

a **Candle**stick **T**echnical analysis tool **L**ite for stock market. ability to use **c++14** even with **simd** and **omp** to edit your technical indicators on this app lite.

K线技术分析工具，在这个平台上你能够使用c++14编写K线技术指标等。
